Table Structure Classes -
-Table main container - sw-table-recipe

This class is the main container class of the table. Use "table" tag when using.

-Tables first row - sw-first-columns-table

First row of the table containing main column headers. Write "tr" element inside "thead"

-First class table row - sw-first-list-item-table

First class item of the table. Use this class in "tr" tag. Clicking opens second class "itemthead"

-Second class table container - sw-detail-second-tr

Use in a "tr" tag.

-Vertical table column - sw-vertical-column-table

Vertical table of the table. Use “td” tag.

-Second class table list item - sw-detail-second-tr-btn

Second class list item class. Use in a “tr” tag. Clicking opens third class table

-Third class table - sw-third-column-heading-table

Third class table. Use in a "tr" tag

-Third class list item - sw-third-list-item-table

Third class table list item. Use in a “tr” tag.
